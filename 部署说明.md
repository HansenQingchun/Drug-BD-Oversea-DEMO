# 启星智谷 · 药物出海评估 DEMO —— 部署到腾讯云指南

本 DEMO 是**纯前端静态网站**（打包后只有 `index.html` + 静态 JS/CSS），
不需要 PHP / MySQL，可放在任意服务器目录、子目录或 iframe 中运行。
打包产物已生成：项目根目录下的 **`drug-demo-dist.zip`**。

---

## 方式一：一键脚本上传（最省事，推荐）

> 前提：你的腾讯云服务器已开启 SSH（22 端口），并知道公网 IP、登录用户名、密码。

在本项目目录打开 PowerShell，运行（把参数换成你自己的）：

```powershell
.\deploy.ps1 -ServerIP 123.45.67.89 -WebRoot /www/wwwroot/你的域名
```

脚本会自动完成「打包 → 压缩 → 上传 → 解压」。中途会提示你输入服务器密码（密码不会被保存）。
完成后访问： `https://你的域名/drug-demo/`

常见参数：
- `-User ubuntu` ：若镜像是 Ubuntu，登录名通常是 `ubuntu`（腾讯云 CVM/轻量默认多为 `root`）。
- `-Subdir pipeline` ：改成别的子目录名，访问地址相应变为 `/pipeline/`。
- `-WebRoot` ：宝塔面板默认网站根目录是 `/www/wwwroot/你的域名`。

> 如果 `scp` 报连接失败：到腾讯云控制台「防火墙/安全组」确认已放行 **22 端口**。

## 方式二：宝塔面板网页上传（不用命令行）

1. 登录服务器的**宝塔面板** → 左侧「文件」。
2. 进入网站根目录，通常是 `/www/wwwroot/你的域名/`。
3. 新建文件夹 `drug-demo` 并进入。
4. 点「上传」，选择本项目里的 `drug-demo-dist.zip` 上传。
5. 上传完成后，右键该 zip → 「解压」到当前目录。
6. 浏览器访问 `https://你的域名/drug-demo/` 即可。

---

## 方式三：嵌入到现有 WordPress 页面（统一站点导航/品牌）

先按方式一或方式二把文件放到 `/drug-demo/`，然后：

1. WordPress 后台 → 「页面」→ 新建页面（如标题「出海管线评估」）。
2. 插入一个「自定义 HTML」区块，粘贴：

   ```html
   <iframe src="/drug-demo/" style="width:100%;height:90vh;border:0;"></iframe>
   ```

3. 发布。该页面就会出现在你的 WordPress 站点里，内部嵌入运行本 DEMO。

## 绑定域名 + 开启 HTTPS（方式四，推荐正式上线用）

> 目标：把现在的 `http://62.234.75.97/drug-demo/` 升级为
> `https://你的域名/drug-demo/`（绿色小锁，证书免费、自动续期）。

### 前提（这两步只能你来做）
1. **有一个域名**（在腾讯云/阿里云/Namecheap 等处购买均可）。
2. **做 DNS 解析**：在域名服务商后台加一条 **A 记录**
   - 主机记录：`@`（或子域名如 `drug`）
   - 记录值：`62.234.75.97`
   - 等待 5~30 分钟生效（可用 `ping 你的域名` 看是否返回该 IP）。
3. 腾讯云轻量「**防火墙**」放行 **80** 和 **443** 端口。

### 一键签发（DNS 生效后执行）
本项目已附带 `server-setup-https.sh`。在本机 PowerShell 依次执行
（每条会提示输入 root 密码）：

```powershell
# 1. 上传脚本到服务器
scp "server-setup-https.sh" root@62.234.75.97:/tmp/setup-https.sh
# 2. 在服务器上执行（把域名/邮箱换成你的）
ssh root@62.234.75.97 "bash /tmp/setup-https.sh drug.你的域名.com you@example.com"
```

脚本会自动完成：检查解析 → 安装 certbot → Let's Encrypt 签发证书 →
写入 Nginx HTTPS 站点（80 自动跳 443）→ 校验并重载。成功后访问：

- `https://你的域名/`            （原 WordPress 主站，若探测到 PHP）
- `https://你的域名/drug-demo/`  （药物出海评估平台）

### 安全设计说明
- 脚本**只新增**一个对应你域名的 server 块，不改动现有
  `default_server`（`wordpress.local`），因此原 `http://IP/drug-demo/` 继续可用。
- 若**没探测到 php-fpm**，新域名仅做静态服务（保证 DEMO 可用），
  绝不写错 PHP 路径拖垮 WordPress。
- 证书 90 天有效，certbot 会装好定时任务自动续期，无需手动维护。

> 备选：也可在**腾讯云控制台**「SSL 证书」免费申请，再在轻量「HTTPS 配置」
> 里一键绑定到该域名——纯网页操作，不用命令行，效果等同。
